GitHubの慣習的なリポジトリ名

GitHub Projectsでタスク管理しようと思って、 タスク管理のためのリポジトリ名で一般的なものはあるのか調べたけれど、 何名かtodosを使っている人は見かけたが、一般的といえるものはななかった。

せっかく調べたので、慣習的といえそうな名前を書いておく。

システム上でサポートされている名前

username.github.io
GitHub Pagesでユーザーサイトを公開するためのリポジトリ
organization.github.io
GitHub PagesでOrganizationサイトを公開するためのリポジトリ
username
ユーザープロフィールREADMEを管理するリポジトリ
Organizationには対応していないようだった
  • GitHub Pagesについて
  • プロフィールのREADMEを管理する

よく使われる名前

dotfiles
ドットファイルを公開するリポジトリ
til
Today I Learned
今日学んだことを書くリポジトリ
ama
Ask Me Anything
ユーザーへの質問用リポジトリ
