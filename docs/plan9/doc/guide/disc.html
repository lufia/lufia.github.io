CDの取り扱い

2009年9月5日更新

CD-Rを焼く

  1. cdfsでマウントする。
  2. 音楽ならwa/、データならwd/にコピー。
  3. 書き込みが終わって、そのディスクに追記しないならwa/またはwd/を削除。

コマンドになおすと以下の通り。

% cdfs
% #echo [quick]blank >/mnt/cd/ctl
% cp data /mnt/cd/wd
% rm /mnt/cd/wd

最近はブルーレイにも対応しているようですね。

CDのマウント

CDから最初のトラックを/n/cdにマウントする場合

% cdfs
% 9660srv -f d000
% mount /srv/9660 /n/cd
